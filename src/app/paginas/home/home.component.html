Conversor de Moedas

Bem-vindo ao projeto Conversor de Moedas, desenvolvido em Angular como parte da matéria Desenvolvimento de Aplicações com Frameworks no 6º período de Sistemas de Informação na Faculdade Metodista Granbery.

Este aplicativo permite a conversão de valores monetários entre diferentes moedas utilizando a API ExchangeRate. Ele foi projetado para listar as taxas de câmbio em tempo real, converter valores entre diferentes moedas e armazenar o histórico de conversões realizadas, que podem ser visualizadas e excluídas.

Funcionalidades

  • Listagem de Moedas: Exibe uma tabela com as taxas de câmbio das moedas disponíveis na API, com ordenação, filtro e paginação.
  • Conversão de Moedas: Permite selecionar moedas de origem e destino, inserir um valor e visualizar o resultado da conversão.
  • Histórico de Conversões: Armazena as conversões realizadas, permitindo ao usuário visualizar e excluir registros.

A API ExchangeRate

A ExchangeRate API é uma das principais fontes para obter taxas de câmbio em tempo real. Utilizamos o endpoint /latest para obter as taxas de câmbio mais recentes a partir de uma moeda base, como o dólar americano (USD).
